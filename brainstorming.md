# Mögliche Ideen für MCP aus dem eigenen Umfeld

## E-Learning Plattform (LMS)

- Kursinhalte
  - Zielgruppe: Studierende
  - AI-Tool: AI-Freund (LibreChat/LiteLLM)
  - Tools:
    - Search
    - Fetch (Metadaten und Kursinhalte)
- Autom. Testing
  - Fertige MCP Server
  - CLIs
- Website-Tests (z.B. Rechtschreibfehler auf Webseite suchen)
- Gebäudeleitsystem
- Integration mit LMS etc.
  - Kurs anlegen, Inhalte hinzufügen, Mails versenden, etc.
